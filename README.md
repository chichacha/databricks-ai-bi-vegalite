# 🚀 databricks-ai-bi-vegalite

> **Learning in public, one visualization at a time.**

A growing collection of **Vega-Lite** examples built specifically for **Databricks AI/BI**.

Whether you're creating your very first KPI card or experimenting with advanced custom visualizations, this repository aims to provide practical, copy-and-modify examples using **synthetic data**—never proprietary datasets.

---

## ✨ What's Inside?

Every example includes:

* 🧩 SQL query
* 🎲 Synthetic dataset
* 🎨 Vega-Lite JSON specification
* 💡 Practical explanation
* 📷 Preview image
* 🔧 Tips for customization

## 🎯 Project Goals

This repository is designed to help you:

* Learn Vega-Lite through practical examples.
* Build beautiful visualizations in Databricks AI/BI.
* Discover useful SQL techniques.
* Create reusable dashboard components.
* Experiment without needing access to production data.
